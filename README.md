## ✨ Features

- Dark terminal aesthetic (OKLCH color tokens)
- Animated particle background (Canvas)
- Scroll progress bar
- Loading screen with animated Git logo
- Cursor glow effect
- Hero terminal with typewriter effect
- Flip cards & animated stat counters
- Git operations tabs with copy button
- Interactive Git workflow visualizer
- Step-by-step GitHub setup guide
- Searchable Git commands cheatsheet
- Git best practices section
- Fully functional in-browser mini terminal
- Responsive design (mobile + desktop)
